# Security Policy

If you discover a security vulnerability (especially regarding location data, local storage databases, or API handling), please do not open a public issue. 

Instead, please contact the maintainers privately so we can address it securely. As a Sadaqah Jariyah project, we aim to provide the most secure and private environment for our users.

## Supported Versions

Only the latest release and the `main` branch are actively supported for security updates. 

## Reporting a Vulnerability

Please email your findings directly to the repository owner. We will endeavor to acknowledge your report within 48 hours. We deeply appreciate your efforts to keep this community project secure.
