# Contributing to hYYa Prayer Pal

## 🚨 STRICT CONTENT-ONLY POLICY 🚨

Thank you for your interest in contributing to the hYYa Prayer Pal repository. Because this project is distributed as a heavily scrutinized Sadaqah Jariyah, **we do not accept code-based Pull Requests or architectural changes.** The application logic, architecture, and feature sets are locked and maintained exclusively by the core team.

Our goal is to expand the religious value of this application. Therefore, we **ONLY** accept authenticated, verified Sunni religious content.

## Permitted Contributions

We welcome and deeply appreciate contributions in the following specific categories:

1. **Quran Translations (Verified):** High-quality, widely accepted translations of the Holy Quran.
2. **Tafsir Translations:** Validated translations of established Sunni Tafsir (e.g., Tafsir Ibn Kathir, Tafsir Al-Jalalayn).
3. **Hisn Al-Muslim (Fortress of the Muslim):** Translations of the supplications into secondary languages.
4. **Quran Audio Reciters:** Links and metadata to Quran audio APIs/repositories that combine Arabic recitation with translated audio.

## Verification Requirement

To protect the integrity of the application, **any submission MUST include the source, publisher, or scholarly reference** of the translation. Submissions lacking verifiable publisher backing to ensure they align with orthodox Sunni scholarship will be rejected.

## Submission Format

Because we do not accept direct code modifications, please submit your data either:

1. **As JSON/XML/Markdown files:** Structure your translations in standard data formats and submit them via a Pull Request isolated entirely to the `public/data/` or relevant content directories.
2. **Via GitHub Issues:** Open a new issue titled `[CONTENT SUBMISSION]` and attach your files or API endpoints, along with the required verification sources. The core team will review the content and inject it into the application logic safely.

May Allah reward you for your efforts to spread beneficial knowledge. 🤲
